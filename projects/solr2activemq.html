At Trulia, i also took care of the integration and release of Thoth with our search infrastructure and APIs.

  • Thoth API : Main technologies, languages: Javascript, Node.js
  • Thoth Dashboard : Main technologies, languages: SASS, Html, Javascript, (NV)D3.js
  • Thoth Machine Learning : Main technologies, languages: Java, H2o, Mallet
  • Thoth Monitor: Main technologies, languages: Java
